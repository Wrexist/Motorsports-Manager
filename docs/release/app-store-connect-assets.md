# App Store Connect packaging checklist

## Store metadata (legal-safe)
- [ ] App name + subtitle contain **no** prohibited trademark terms (see `docs/legal/naming-disclaimer.md`)
- [ ] Promotional text + description + “What’s New” reviewed with the same rules
- [ ] Keywords reviewed (no trademark bait)

## Visual assets
- [ ] App icon (adaptive rules satisfied)
- [ ] Screenshots for required device sizes you support
- [ ] Optional App Preview video

## Support + compliance URLs
- [ ] Support URL (can be a simple site or mailto policy — must be valid)
- [ ] Privacy policy URL (must match actual behavior)

## Review information
- [ ] Notes for reviewer include: how to progress, sandbox IAP steps (if any), anything requiring hardware
